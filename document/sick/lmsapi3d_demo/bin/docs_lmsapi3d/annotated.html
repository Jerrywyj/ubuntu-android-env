LMSAPI3D Estructura de datos

Lista de estructuras con una breve descripción:
LMSAPI::LMSAPI_CONNECTIONClase envoltorio para administrar una conexión al sensor
LMSAPI::LMSAPI_DATAClase envoltorio para contener los datos de medición
LMSAPI::LMSAPI_LIBClase Envoltura para encapsular las funciones de la libreria dinamica LMSAPI
LMSAPI_csharp::LaserData3DClase para almacenar mediciones sucesivamente durante un barrido vertical. Esto es útil para medir sólidos. También tiene funciones para la representación de los datos en 3D. Para iniciar una serie de mediciones, utilice begin_capturing y luego capture_data
LMSAPI_csharp::LaserDataSceneNodeClase para representar una lectura de datos en una malla 3D
LMSAPI_csharp::LaserViewer3DClase para manejar una escena Irrlicht
LMSAPI_csharp::LMSAPIControlClase para adminstrar las propiedades de conexión y la captura del sensor. Contiene una copia offline de los parámetros de medición y almacena los objetos de conexión (LMSAPI_CONNECTION) y de lectura de datos (LMSAPI_DATA)
