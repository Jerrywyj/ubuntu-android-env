Descripción detallada

Clase envoltorio para contener los datos de medición.

LMSAPI_CONNECTION newconnection = LMSAPI_LIB.create_connection_object(1,100,50,8,0); newconnection.m_scale_distance = 1000.0f;//Configurando la escala a metros. LMSAPI_DATA data = newconnection.request_measurement();//obteniendo los datos de medición

Utilice un objeto LMSAPI_CONNECTION para crear este objeto de datos de medición
