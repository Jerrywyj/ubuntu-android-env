Descripción detallada

Clase para almacenar mediciones sucesivamente durante un barrido vertical. Esto es útil para medir sólidos. También tiene funciones para la representación de los datos en 3D. Para iniciar una serie de mediciones, utilice begin_capturing y luego capture_data.
