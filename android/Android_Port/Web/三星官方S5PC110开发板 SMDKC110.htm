三星官方S5PC110开发板 SMDKC110

 

The SMDKC110(S5PC110 Development Kit) is the official reference/evaluation board for Samsung S5PC110.

It is the basic system-based hardware design which uses the S5PC110.

It can evaluate the basic operations of the S5PC110 and assist in developing codes.
